# BudgetBuddy

BudgetBuddy is a simple web app that helps users build a safer monthly budget and plan for a secure financial future.

## Features
- Enter income, fixed expenses, variable expenses, and current savings.
- Generate a safety-focused monthly budget plan.
- Get a recommended split for leftover cash:
  - 50% to emergency fund
  - 30% to future goals
  - 20% to lifestyle buffer
- Estimate months needed to reach an emergency fund target.
- See warnings when the budget is risky.

## Run locally
Because this is a static app, you can open `index.html` directly or run a simple local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
